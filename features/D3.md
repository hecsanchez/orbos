Step 2 — Database Schema
Create apps/api/src/db/schema.ts using Drizzle ORM with all 7 tables exactly as defined in context.md. Key requirements:

Enable the pgvector extension: CREATE EXTENSION IF NOT EXISTS vector
The standards table must have an embedding column of type vector(1536) — leave it nullable for now, it will be populated on Day 4
The attempts table must have source as an enum: lesson | phenomenon
The mastery_state table must have has_direct_lesson_attempt as a boolean defaulting to false
Use UUIDs for all primary keys, generated by the DB with gen_random_uuid()
All timestamps default to now()

Also create apps/api/src/db/index.ts — the Drizzle client connected to PostgreSQL via env var DATABASE_URL.
