# D19 — E2E Test: 5-Day Simulated Week

Sprint D25: End-to-end test simulating a 5-day learning week for 3 student profiles (ages 5, 8, 11).

## Changes

### Vitest Integration Test
- **Created** `apps/api/src/agents/orchestrator/five-day-simulation.spec.ts`
  - 15 tests covering full pipeline simulation
  - 3 student profiles: Ana (5yo, grade 1), Miguel (8yo, grade 3), Sofia (11yo, grade 5)
  - Each profile gets 6 standards across Matematicas, Espanol, Ciencias

#### Per-Student Tests (x3):
- **5-day mastery progression**: simulates 5 days of attempts with increasing correctness (40% → 90%), verifies mastery trends upward, at least one standard reaches ≥ 0.5
- **Plan type adaptation**: verifies 0.0 mastery → lesson, 0.65 mastery → practice, 0.9 mastery → skipped
- **Standards reduction**: verifies mastered standards drop from candidate pool

#### Cross-Student Verification:
- **Break insertion**: no session segment exceeds 35 min without a break
- **Mastery boundaries**: 0.8 + lesson → advance, 0.8 without lesson → practice, < 0.5 → reteach
- **Phenomenon weighting**: evidence attempts weighted 0.8x vs lesson attempts
- **Hint penalty**: hints reduce mastery score
- **Speed bonus**: fast correct answers score higher
- **Recency bias**: improving patterns score higher than declining patterns

### Live API E2E Test
- **Created** `e2e/api-e2e/src/api/five-day-simulation.spec.ts`
  - Jest test for running against live API server
  - Tests: health check, student fetch, plan generation, plan caching, attempt logging, mastery progression, API endpoint availability (standards, lessons, safety logs, phenomena)
