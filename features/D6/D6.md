Create apps/ipad/src/services/tts.service.ts:
typescriptimport * as Speech from 'expo-speech';

export class TTSService {
// Speaks text in es-MX locale
// Stops any currently speaking utterance before starting
speak(text: string): Promise<void>

// Stops current speech immediately
stop(): void

// Returns true if currently speaking
isSpeaking(): Promise<boolean>
}
Requirements:

Always use language: 'es-MX'
Default rate: 0.9 — slightly slower than normal for children
Default pitch: 1.0
Export a singleton instance — components import the instance, not the class
If expo-speech is unavailable (simulator quirk), fail silently and log a warning — never crash the app

The attempt_queue table is critical — define it carefully:
sqlCREATE TABLE IF NOT EXISTS attempt_queue (
id TEXT PRIMARY KEY,
student_id TEXT NOT NULL,
standard_id TEXT NOT NULL,
interaction_component TEXT NOT NULL,

time_spent_seconds INTEGER NOT NULL,
hint_used INTEGER NOT NULL,      -- 0 or 1
source TEXT NOT NULL,            -- 'lesson' | 'phenomenon'
