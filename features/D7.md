export class SafetyAgent {

}
The Safety Agent must check for all of the following:
Rule-based checks (no LLM needed — fast, deterministic):

All component names are in the approved list — reject immediately if not
No interaction block exceeds the word limit for its component type
tts_text is present on every block that requires it
No URLs, external links, or domain names in any text field
No personally identifiable information patterns (email, phone number formats)

LLM-based checks (one LLM call per script):

Age-appropriate tone and vocabulary for the target age
Content is on-topic for the linked SEP standard
No sensitive, violent, or inappropriate content
Positive and encouraging framing — no negative language directed at the child

Requirements:
